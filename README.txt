criar o projeto 
npm create vite@latest

instalar
yarn ou npm i

instalar dependencias da interface declarativa
yarn add postcss autoprefixer -D
yarn add tailwindcss init -p
npx tailwindcss init -p

dependencias
yarn add @apollo/client graphql
yarn add phosphor-react
yarn add date-fns
yarn add @vime/core @vime/react
yarn add react-router-dom
yarn add classnames
yarn add @graphql-codegen/typescript @graphql-codegen/typescript-operations @graphql-codegen/typescript-react-apollo -D
yarn add @graphql-codegen/cli -D

rodar como desenvolvedor
yarn run dev
